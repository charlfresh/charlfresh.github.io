This is a lesson on binary!

What is binary?

Binary is the language that computers use to understand information, there are only two digits in the binary language, 1 and 0.

Using combinations of 1 and 0 computers are able to represent and understand very simple to very complex information.

Each digit in binary (either 1 or 0) at the microsocpic level, is just a switch that is either on or off.

How is the decimal number system represented using binary?

Binary can easily be converted into decimal (0-9) using a simple conversion method.

Each digit in a binary number sequence, such as 10111001 holds the value of a power of 2.

binary numbers read right too left, the first digit is on the far right.

For example, the first digit in a binary number represents 2^0 (1), the second digit represents 2^1 (2), and so on.

However the value will only be held if the binary number in that digit is a 1.

How are english characters represented using this system?

The system used to convert 8 digit binary numbers into english characters is called ASCII.

The ASCII system is used to convert any character into binary, so the conversion table is very large.
